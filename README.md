# 🧩 Zestora - E-commerce CMS Dashboard 

## 📌 Overview
A scalable and dynamic Content Management System (CMS) built for managing an e-commerce platform. This admin dashboard enables efficient management of products, users, and application content with a focus on modular architecture and real-world business workflows.

---

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS
- JavaScript (ES6+)

---

## ✨ Features
- 🔐 Authentication & role-based access control
- 👥 User management (Add / Edit / Delete users)
- 📦 Product management system (CRUD operations)
- 📊 Dashboard analytics and insights
- 🔍 Search, filtering, and sorting functionality
- ⚙️ Dynamic page/content management
- 🔄 Global state management using Redux Toolkit
- 📱 Fully responsive admin interface
- 🌙 Dark mode support

---

## 🧠 Key Learnings
- Designing scalable CMS architecture
- Implementing role-based access control (RBAC)
- Managing complex global state using Redux Toolkit
- Building reusable UI components for dashboards
- Handling real-world admin workflows and data flows
