# 🧑‍💻 Content Management System (CMS)

## 📌 Overview
A dynamic CMS dashboard that allows admin users to manage pages, routes, and content with role-based access control.

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS

## ✨ Features
- 🔐 Role-based authentication
- 📄 Dynamic page creation
- 🧭 Dynamic routing
- ⚙️ Admin dashboard UI
- 📊 Structured content management

## 🧠 Key Learnings
- Role-based routing
- Dynamic UI rendering
- Scalable architecture design

## 🚀 Future Improvements
- Backend integration
- Content versioning
- Analytics dashboard
